Compiler for Coda MIDI description language

## Installation

## Lexical Specification
### Tokens
**Note** - [A-F]*[#, b, n]?[1-7] \
**Chord** - [A-F] \
**Declarator** - ['!'] \
**Separator** - [','] \
**Connector** - ['>']['>']? \
**Lbrace** - ['{'] \
**Rbrace** - ['}'] \
**Lbracket** - ['['] \
**Rbracket** - [']'] \
**Separator** - [',']
